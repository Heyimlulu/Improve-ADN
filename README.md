# ADN Improver

Coming soon to Chrome Web Store, Edge Add-ons and Firefox Add-ons.

## Manual/Development installation instructions

1. Clone this repository.
2. In Chrome, go to chrome://extensions.
3. In the top right corner, click the "Developer Mode" checkbox.
4. Click the now-visible "Load unpacked" button. Navigate to the directory where you just cloned the repository.
5. The ADN Improver extension should now be visible in your extensions list.
